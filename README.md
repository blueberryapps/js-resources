# JavaScript Developer Resources

This page describes our platform foundation - libraries and concepts we use in our daily work on JavaScript apps.

## Libraries and frameworks

#### React
A javascript library for building user interfaces.

- [React.js website](https://facebook.github.io/react/)
- [React.js documentation](https://facebook.github.io/react/docs/getting-started.html)
- [ReactJS For Stupid People](http://blog.andrewray.me/reactjs-for-stupid-people/)
- [You’re Missing the Point of React](https://medium.com/@dan_abramov/youre-missing-the-point-of-react-a20e34a51e1a#.skuzwqtrj)
- [Using React is a Business Decision, Not a Technology Choice](https://blog.formidable.com/using-react-is-a-business-decision-not-a-technology-choice-63c4641c5f7#.z6g637rqt)
- [Sebastian Markbage: Minimal API Surface Area [JSConf EU 2014]](https://www.youtube.com/watch?v=4anAwXYqLG8)

#### Este.js
Platform foundation, dev stack and starter kit for functional and universal (browser, server, mobile) React apps.

- [Este.js github](https://github.com/este/este)

#### Redux
Predictable state container for JavaScript apps.

- [Redux documentation](http://redux.js.org)
- [Getting Started with Redux [Egghead.io course]](https://egghead.io/series/getting-started-with-redux)

#### react-router
react-router is a complete routing library for React.

- [react-router github](https://github.com/rackt/react-router)
- [react-router documentation](https://github.com/rackt/react-router/blob/master/docs/README.md)

#### Immutable.js
Immutable collections for JavaScript.

- [Immutable.js website](http://facebook.github.io/immutable-js/)
- [Immutable.js documentation](http://facebook.github.io/immutable-js/docs/)

#### Bluebird.js
Bluebird is a full featured promise library with unmatched performance.

- [Bluebird.js website](http://bluebirdjs.com/docs/getting-started.html)
- [Bluebird.js documentation](http://bluebirdjs.com/docs/api-reference.html)

#### Fetch API
Fetch API allows you to make network requests similar to XMLHttpRequest (XHR). The main difference is that the Fetch API uses Promises.

- [Fetch polyfill from GitHub](https://github.com/github/fetch)
- [isomorphic-fetch](https://github.com/matthew-andrews/isomorphic-fetch) Universal Fetch library built on top of GitHub's Fetch polyfill

#### Radium
Set of tools to manage inline styles on React elements.

- [Radium website](http://projects.formidablelabs.com/radium)
- [Radium github](https://github.com/FormidableLabs/radium)

#### Autoprefixer
Parse CSS and add vendor prefixes to rules by Can I Use.

- [Autoprefixer github](https://github.com/postcss/autoprefixer)

#### Express.js
Fast, unopinionated, minimalist web framework for Node.js.

- [Express.js website](http://expressjs.com)
